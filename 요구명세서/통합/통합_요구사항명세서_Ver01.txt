요구 사항 명세서

1. 경매 임인규님 VER 요구사항명세서
	1) 경매는 1000천원부터 시작한다
	2) 입찰단위는 100,200,500,1000, 2000, 5000,10000,20000,50000,100000원으로 가능하다
	3) 경매 기간은 3일로 시작하고 10일까지 가능하다
	4) 입찰이 안되면 연장경매도 가능하다. 단 1회만 가능
	5)경매가 유찰될경우 1회에 한해 자동으로 경매가 재진행된다

2. 물품 등록
	1) 문의, 입찰취소요청, 내 물품 고객평가, 보유 판매 예치금
	2) 반품은 불가하다
	3) 판매자는 품목분류, 상품명, 첨부파일, 상품설명, 배송방법을 등록한다
	4) 품목분류는 대분류, 중분류, 소분류, 세분류로 나뉜다
	5) 상품명은 최대 50자이다
	6) 첨부파일은 최대 3개까지 첨부하고, 용량은 최대 10Mb를 넘지않는다
	7) 상품설명은 500자까지 가능하다.
	8) 배송방법은 택배와 직접전달로 한다. 둘 중에 하나만 선택 가능하다.
	9) 입금계좌는 1개만 등록한다

3. 물품 정보

- [ ]  상품 정보(간단)
    - [ ]  `비회원`과 `회원`이 상품을 확인 가능하다.
    - [ ]  `비회원`은 `입찰하기`와 `관심상품`을 선택하면 로그인 페이지가 팝업창으로 보여진다.
    - [ ]  `판매자`가 등록한 이미지와 상품 정보를 보여준다
    (상품등록 페이지 내용과 관련)
    - [ ]  `회원`은 입찰하기 버튼이 활성화 되어있는 상태에서 입찰이 가능하다.
    - [ ]  입찰 수는 입찰할 때마다 1씩 증가한다.
    - [ ]  `회원`은 `경매기록`을 볼 수 있다.
    - [ ]  `남은시간`은 시간-분-초로 확인이 가능하며, 종료시간(년-월-일 시:분)과 함께 표시한다.
    - [ ]  `경매기록`은 `시간순`으로  `내림차순`으로 보여준다.
    - [ ]  경매가 종료가 되면 `남은 시간`을 경매종료 텍스트로 보여준다.
    - [ ]  `판매자`는 지정한 `마감연장`을 할 수 있다.
    - [ ]  입찰 종료 전에는 `입찰하기`버튼이 활성화 된다.
    - [ ]  입찰 종료 후에는 `입찰하기`버튼이 `경매기록`버튼으로 변경된다.
    - [ ]  입찰 종료 후에는 `상품 종료 메시지`를 보여준다.
- [ ]  상품 정보(상세)
    - [ ]  `상품정보`와 `판매자가 받은 구매후기`, `상품문의`는 탭을 통해 페이지 전환이 가능하다.(화면 이동X)
    - [ ]  `판매자`가 등록한 `이미지`와 `내용`을 보여준다.
    - [ ]  `상세정보 더보기`를 버튼 클릭하면 상품 정보를 전체 보여준다.
    - [ ]  `상세정보 더보기`가 활성화 되면 `상세정보 접기`로 버튼이 변경되고 클릭하면 상품 정보를 일부만 보여주다.
    - [ ]  직거래 하는 경우 피해 발생 시 책임지지 않는다는 주의사항을 명시해둔다.
    - [ ]  `상품 정보 제공 공시`에 `상품번호` (… 상품등록 관련 협의)
- [ ]  판매자가 받은 구매 후기
    - [ ]  `판매자`가 판매했던 제품의 `이름`과 `후기`를 표시한다.
    - [ ]  `구매자`만 후기 작성이 가능하다.
    - [ ]  별 5개 만점에 후기 등록 별점으로 표시한다.(ex. 1/5)
    - [ ]  `아이디`일부만 표시하고, 후기 등록한 날짜를 표시한다.
    - [ ]  *후기: 글자수 지정해야함 (한줄평 생각)
    - [ ]  한 페이지당 후기 5개로 표시하고
    - [ ]  페이지당 5개로 `시간순`으로  `내림차순`으로 보여준다.
- [ ]  상품 문의 (→ 보류)
    - [ ]  쪽지 보내기?
- [ ]  관심상품 등록
    - [ ]  `비회원`일 경우 로그인 페이지가 팝업창으로 보여준다.
    - [ ]  `회원`일 경우 관심상품 등록하기 서브메뉴로 보여주며 지정한 폴더에 저장이 가능하다.
    - [ ]  `분류선택`과 `메모`로 관심상품 등록을 할 수 있다.
    - [ ]  `분류선택`은 `분류없음`폴더가 디폴드 값이다.
    - [ ]  `회원`은 `새 분류 등록`을 통해서 폴더 등록 및 취소가 가능하다.
    - [ ]  `메모`는 한글, 영문, 숫자, 특수문자 포함 30byte로 지정한다.
    - [ ]  `회원`은 관심상품 등록 하기 전에 `저장`만 할지, `관심상품목록`으로 이동을 할지 선택이 가능하다.
- [ ]  경매 기록 조회
    - [ ]  입찰 상품에 `상품명`, `경매마감일자` , (`최대 구매가능 수량`, `판매지역`) 조회한다.
        
        (괄호는 협의할 내용 → 상품등록 페이지)
        
    - [ ]  `입찰자ID` `입찰일자` `입찰가격` `수량` `누적수량` 을 조회한다.
    - [ ]  `입찰자ID` 는 아이디 일부만 표시한다.
    - [ ]  `입찰일자` 는 년-월-일 시:분:초:프레임으로 표시한다.
    - [ ]  `입찰가격`은 원 단위로 표시한다.
    - [ ]  `수량`은 개로 표시하고, `사용자`가 구매한 수량을 표시한다.
    - [ ]  `누적수량`은 입찰 종료 후 `낙찰자`에게만 표시한다.
    - [ ]  입찰 종료 전에는 `낙찰 가능자`를 확인 가능하다
    - [ ]  입찰 종료 후에는 `낙찰자`를 확인 가능하다.

4. 마이페이지 - 구매

	4-1. 구매자

	구매/경매 내역
		=> 구매자는 구매 내역을 통해 구매 물품명, 낙찰가, 물품 수량, 구매일, 배송일, 물품 번호을 알 수 있습니다.
		=> 구매자는 기간별로 1개월/3개월/6개월/9개월/1년치 데이터를 조회할 수 있습니다.
		=> 구매자는 해당 데이터를 한 페이지에 10건/30건 씩 데이터를 조회할 수 있습니다.
		=> 구매자는 구매 취소 내역을 통해 물품 번호, 구매 물품명, 물품 가격, 물품 수량, 구매일, 취소일, 취소 사유를 알 수 있습니다.
		=> 구매자는 경매 참가 내역을 통해 경매 번호, 물품 명, 입찰 가격, 낙찰 가격, 낙찰 여부, 경매 시작일, 경매 종료일, 경매 개최자, 경매 종류를 알 수 있습니다.

	후기 내역 (작성자는 외래키)
 	후기 작성	
		=> 구매자는 경매가 종료하고 상품 배송이 완료되면 해당 상품의 후기를 작성할 수 있습니다.
		=> 후기는 별 0 ~ 5 개를 넣을 수 있습니다.
		=> 후기에 대한 3 ~ 4줄 내용을 작성할 수 있습니다.
 	후기 조회
		=> 구매자는 구매자가 작성한 후기를 조회할 수 있습니다.
		=> 구매자는 상품/판매자 별로 작성한 후기를 조회 할 수 있습니다.
		=> 구매자는 기간별로 1개월/3개월/6개월/9개월/1년치 데이터를 조회할 수 있습니다.
 	후기 수정
		=> 구매자는 해당 후기의 별의 개수를 수정할 수 있습니다.
		=> 구매자는 해당 후기의 내용을 수정 할 수 있습니다.
 	후기 삭제
		=> 구매자는 해당 후기를 삭제 할 수 있습니다.

	4. 기타 기능 (정태환님 VER)
		4-1 구매자 
		입찰신청 기능
			=> 구매자는 원하는 상품의 경매 참가를 할 수 있는 권한을 획득 할 수 있다.
		판매자 찜 기능
			=> 구매자는 자신이 선호하는 판매자를 선택 할 수 있다.
		상품 좋아요 기능
			=> 구매자는 자신이 선호하는 상품을 선택할 수 있다.
		공유 기능
			=> 구매자는 해당 상품의 경매 정보, 해당 상품의 판매자 정보, 해당 상품 정보를 다른 사람에게 전달 할 수 있다. 
		판매자 신고기능
			=> 구매자는 불량 판매자를 신고 할 수 있습니다.
			=> 구매자는 판매자명, 판매자 신고 사유를 작성해서 해당 판매자를 신고 할 수 있습니다.
		시세조회 기능
			=> 구매자는 해당 상품의 실시간 시세를 조회할 수 있습니다.
		판매자 상품 조회기능
			=> 구매자는 해당 판매자가 판매하는 상품을 조회할 수 있습니다.
			=> 해당 조회 정보는 판매자명, 판매 상품명, 판매가(=입찰가), 판매 시작일, 경매 타입을 알 수 있습니다.
		판매자 신용조회기능
			=> 구매자는 해당 판매자의 사이트의 신용 정보를 조회 할 수 있습니다.
			=> 해당 신용 정보는 판매자명, 판매자의 신고 건수, 신고 점수를 알 수 있습니다.
		입찰포기 기능
			=> 구매자는 해당 상품의 경매 입찰을 포기 할 수 있습니다.
		흥정 기능(+@)
			=> 경매중에 돈으로 더 못 올리는 입찰자가 현재 입찰가+@(물건)으로 판매자한테 요청합니다.
		4-2 판매자
		구매자 제한기능
			=> 판매자는 해당 상품의 경매 참가자 중에 원하지 않는 경매 참가자 OR 원하지 않는 회원 등급의 경매 참가자를 경매에 참가할 수 없게 할 수 있습니다.
		시간 연장기능
			=> 판매자는 해당 경매의 시간을 연장할 수 있습니다.

4-3 공통(구매자/판매자)

회원가입

=> 비회원은 아이디, 비밀번호, 비밀번호 확인, 이메일, 생년월일, 집 주소, 핸드폰 번호를 입력하여 회원가입한다.

=> 아이디는 아이디 중복 체크를 해야 한다.

=> 비밀번호는 저장 시 비밀번호를 암호화하여 저장한다.

=> 이메일은 이메일 형식에 맞게 작성한다.

=> 생일은 년 월 일 순으로 입력한다. EX) 1990-01-01

=> 생년월일을 클릭하면 달력을 이용하여 생년월일을 선택한다.

=> 집 주소는 우편번호를 입력한다.

=> 집 주소는 도 , 시, 구/동, 읍/면/리를 입력한다

=> 핸드폰 번호는 핸드폰 형식에 맞게 작성한다.

=> 비회원은 비회원 가입 후 이메일 인증을 통해 계정을 활성화 해야 사용이 가능하다.

=> 이메일 인증 번호는 영어, 숫자를 이용한 6글자를 링크 형식으로 이메일로 전송한다.

=> 입력한 정보가 잘못된 경우 안내 메세지를 출력한다(유효성 검사)

=> 아이디, 비밀번호, 비밀번호 확인, 이메일, 생일은 필수 항목이다.

로그인

=> 회원은 아이디, 비밀번호를 입력하여 로그인을 한다.

=> 회원은 자동 로그인을 선택하여 자동 로그인을 한다.

=> 로그인 실패시 안내 메세지를 출력한다.

채팅

=> 구매자와 판매자는 채팅 기능을 통해 서로간의 의사소통을 할 수 있다.

=> 구매자는 경매 도중에 판매자에게 문의 하고 싶은 점을 채팅을 통해 문의 할 수 있다.

=> 판매자는 경매 도중에 구매자의 문의 사항을 답변할 수 있다.

=> 구매자/판매자는 채팅 기능을 통해 현재 시간 : 입찰가 : 입찰자의 정보를 알 수 있다.
