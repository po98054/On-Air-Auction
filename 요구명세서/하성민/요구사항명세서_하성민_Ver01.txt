0. 구매자

0-1 구매/경매 내역

=> 구매자는 구매 내역을 통해 구매 물품명, 낙찰가, 물품 수량, 구매일, 배송일, 물품 번호을 알 수 있습니다.

=> 구매자는 기간별로 1개월/3개월/6개월/9개월/1년치 데이터를 조회할 수 있습니다.

=> 구매자는 해당 데이터를 한 페이지에 10건/30건 씩 데이터를 조회할 수 있습니다.

=> 구매자는 구매 취소 내역을 통해 물품 번호, 구매 물품명, 물품 가격, 물품 수량, 구매일, 취소일, 취소 사유를 알 수 있습니다.

=> 구매자는 경매 참가 내역을 통해 경매 번호, 물품 명, 입찰 가격, 낙찰 가격, 낙찰 여부, 경매 시작일, 경매 종료일, 경매 개최자, 경매 종류를 알 수 있습니다.

0-2 후기 내역 (작성자는 외래키)

0-2-1 후기 작성

=> 구매자는 경매가 종료하고 상품 배송이 완료되면 해당 상품의 후기를 작성할 수 있습니다.

=> 후기는 별 0 ~ 5 개를 넣을 수 있습니다.

=> 후기에 대한 3 ~ 4줄 내용을 작성할 수 있습니다.

0-2-2 후기 조회

=> 구매자는 구매자가 작성한 후기를 조회할 수 있습니다.

=> 구매자는 상품/판매자 별로 작성한 후기를 조회 할 수 있습니다.

=> 구매자는 기간별로 1개월/3개월/6개월/9개월/1년치 데이터를 조회할 수 있습니다.

0-2-3 후기 수정

=> 구매자는 해당 후기의 별의 개수를 수정할 수 있습니다.

=> 구매자는 해당 후기의 내용을 수정 할 수 있습니다.

0-2-4 후기 삭제

=> 구매자는 해당 후기를 삭제 할 수 있습니다.

0-3 신고 내역

0-4 배송 조회

1. 판매자

1-1 판매/경매 내역

=> 판매자는 판매 내역을 통해 판매 물품 번호, 판매 물품명, 판매 물품 수량, 판매가, 판매 날짜, 구매 날짜, 판매자명, 구매자명을 알 수 있습니다.

=> 판매자는 기간별로 1개월/3개월/6개월/9개월/1년치 데이터를 조회할 수 있습니다.

=> 판매자는 해당 데이터를 한 페이지에 10건/30건 씩 데이터를 조회할 수 있습니다.

=> 판매자는 경매 개최 내역을 통해 경매 번호, 물품명, 경매 종류, 경매 시작일, 경매 종료일, 입찰 가격, 낙찰 가격, 낙찰자명을 알 수 있습니다.

2. 공통 (구매자/판매자)

2-1 문의사항 내역

2-2 회원 정보

2-2-1 회원 정보 조회

=> 회원 아이디, 비밀번호, 이메일 주소, 핸드폰 번호, 집 주소, 회원 등급, 본인 인증 여부를 조회 할 수 있습니다.

2-2-2 회원 정보 수정

=> 회원 비밀번호를 정확하게 입력하면 회원 정보를 수정 할 수 있습니다.

=> 회원 비밀번호, 이메일 주소, 핸드폰 번호, 집 주소, 본인 인증 여부를 수정할 수 있습니다.

2-3 회원 탈퇴

=> 회원 비밀번호를 정확하게 입력하면 회원을 탈퇴 할 수 있습니다.

경매 취소

=> 판매자는 해당 경매를 취소 할 수 있습니다.

=> 해당 경매를 취소하기 위해서는 경매 취소 사유를 반드시 적어야 합니다.

구매 취소

=> 구매자는 구매자가 구입한 상품의 구매를 취소할 수 있습니다.

경매의 옵션 기능

=> 판매자는 경매의 옵션을 설정할 수 있습니다.

=> 실시간 경매인 경우 입찰자가 입찰가를 부르고 몇초의 제한시간을 둘지를 설정할 수 있습니다. (EX: 10/30/60초)

=> 시간 경매인 경우 판매자는 몇일의 제한 기간을 둘지를 설정할 수 있습니다. (예: 1일/3일/한달)

=> 제한 인원 경매인 경우 판매자는 구매자의 등급별로 제한을 둘 수 있습니다.

=> 판매자는 농수산물 경매인 경우 각 상품의 수량을 설정할 수 있습니다.
